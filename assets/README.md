# Project Assets

This directory contains all supplementary materials generated for the project.

- **/images/**: General images, concept art, metaphysical diagrams.
- **/figures/**: Formal figures, plots, and graphs generated by the simulations in `/Simulations/MASTER_SIMULATION_SUITE/` for inclusion in the manuscripts.
- **/data/**: Any non-code datasets, CSVs, or supplementary data files.
